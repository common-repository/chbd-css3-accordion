== Description ==

This plugin will help users to create an eye catching accordion style with pure css3.

Live Demo: http://www.arefin.me/plugins/chbd-css3-accordion/

Plugin Features

* Shortcode is Enabled.
* Wordpress Custom Post is Enabled.
* No html, css or php code is needed to know.
* Shortcode is generated with TinyMCE Small Button.
* Small in Size.
* Many More..

== Installation ==

= Using The Wordpress Dashboard =

1. First go to the 'Add New' Plugin Dashboard
2. Select `chbd-css3-accordion.zip` file from your computer
3. Simply Upload
4. Activate the plugin via the Plugin Section of WordPress Dashbaord
5. As this plugin supports custom post, you will see a menu in admin panel named 'Accordion'. You can manage all your accordions there. 
6. After adding some accordion title and content, you can just use the shortcode `[chbd_accordion]`

= Using FTP =

1. Extract `chbd-css3-accordion.zip` to your computer
2. Upload the `chbd-css3-accordion` directory to your `wp-content/plugins` directory
3. Activate the plugin on the WordPress Plugin Dashbaord
4. As this plugin supports custom post, you will see a menu in admin panel named 'Accordion'. You can manage all your accordions there. 
5. After adding some accordion title and content, you can just use the shortcode `[chbd_accordion]`

== Frequently asked questions ==

= Do I need to know any coding to setup and use the plugin? =

No, you do not need to know any kind of coding to setup this plugin. You can easily do that just clicking on the TinyMCE or post editor shortcode icon.
